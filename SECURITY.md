# Security Policy

## Supported Versions

Please ensure that the version you wish to report a vulnerability in is currently supported.
You can get an overview of this information in the table.

| Version | Supported          |
| ------- | ------------------ |
| 1   | :white_check_mark: |

## Reporting a Vulnerability

Email joel@webrelics.xyz to report vulnerabilities. The fix is expected to be delivered within a maximum of 5 days.
If your report is denied, you will be notified of our decision and the reason why.
